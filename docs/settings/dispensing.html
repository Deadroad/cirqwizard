Settings - Dispensing

Needle

Parameter Meaning Default value Units
Needle diameter Default needle diameter 0.4 mm

Timing

Parameter Meaning Default value Units
Prefeed pause Duration of pause inserted after the valve is opened and before needle starts to move 0.02 sec
Postfeed pause Duration of pause inserted after the needle is finished dispensing and before it is retracted 0.2 sec
Bleeding duration Duration of valve opening on bleeding screen 0.5 sec

Feed

Parameter Meaning Default value Units
Feed Default feed (speed with which needle moves during dispensing) 100 mm/min

Heights

Parameter Meaning Default value Units
Clearance Default value of clearance – the height above the board, the program uses to move the needle between pads 5 mm
Z offset Z axis offset – Z coordinate in machine coordinate system when needle is slightly above laminate mm
Working height Height of the needle relative to laminate surface during dispensing 0 mm
All heights are specified in work coordinate system (i.e. taking into account Z offset).
