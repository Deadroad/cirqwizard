Settings - Drilling

Tool

Parameter Meaning Default value Units
Feed Default feed on Z axis (speed with which drill penetrates the laminate) 200 mm/min
Speed Value of speed parameter passed to the machine controller for milling operation (PWM width of control pulse in case of Cirqoid machines) 1390 ms

Heights

Parameter Meaning Default value Units
Clearance Default value of clearance – the height above the board, the program uses to move the drill between holes 5 mm
Safety height Default value of safety height – the height above the board, where drill switches from rapid descend to feed 2 mm
Z offset Z axis offset – Z coordinate in machine coordinate system when inserted drill is slightly above laminate mm
Working height The depth of the hole (this should be slightly more than laminate width, to take into account angle at the end of the drill) -2 mm

All heights are specified in work coordinate system (i.e. taking into account Z offset).
