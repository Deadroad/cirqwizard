Settings - Machine

Calibration parameters

Parameter Meaning Default value Units
Y axes difference Difference between zero points of machine’s Y axes mm
Reference pin X X coordinate of the reference pin mm
Reference pin Y Y coordinate of the reference pin mm

Reference pin is a registration pin located in lower left corner of PCB.

Reference pin distances

Parameter Meaning Default value Units
Small laminate Distance between the centers of the registration holes on the smaller of supported laminates along the X axis 65 mm
Large laminate Distance between the centers of the registration holes on the larger of supported laminates along the X axis 90 mm
